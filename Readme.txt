Long homework project i made with a partner in the university.
We take an example of a store that sell Cheese. So, you have the option to make a sell (client chose an cheese and the quantity)
then the program ask you if you want to keep buying, and also if you want to check the chart.
If there is not stock of a cheese the program will promp you "there is no stock", and the way to re-stock is when a week pass.
Sunday's you could check the total amount you made in all the week, and next monday program restart the money in 0.
